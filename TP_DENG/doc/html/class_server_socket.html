Detailed Description

TCP/IP Socket Server. Note: this class supports AF_INET connections following the IPv4 Internet protocol. Other families, such as AF_INET6 or AF_UNIX are not yet supported.

Constructor & Destructor Documentation

ServerSocket::ServerSocket ( )

creates a new ServerSocket. Creates a listening AF_INET socket (using the IPv4 Internet protocol) that waits for connection requests by TCP/IP (SOCK_STREAM) client sockets.
